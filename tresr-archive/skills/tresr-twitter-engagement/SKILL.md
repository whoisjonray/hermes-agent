## CRITICAL: Environment Setup
- **Always load .env with**: `require('dotenv').config({override: true})`
- System env has @jonray keys. `.env` has @0xTRESR keys. Without `{override: true}`, dotenv silently uses wrong account.
- **For OAuth signatures with special chars** (`$`, `'`, `(`, `)`, `!`, `*`): use RFC 3986 encoding:
  ```javascript
  function rfc3986(str) {
    return encodeURIComponent(str).replace(/[!'()*]/g, c => '%' + c.charCodeAt(0).toString(16).toUpperCase());
  }
  ```
- **v1.1 update_profile**: send params as **query string**, NOT form body
